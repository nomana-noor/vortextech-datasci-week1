# vortextech-datasci-week1
# VortexTech Week 1 - Data Cleaning and Visualization

## What I Built
This project analyzes the Amazon Sales Dataset from Kaggle. I cleaned the data (handled missing values, fixed data types) and created 5 visualizations to explore patterns in product categories, pricing, and ratings.

## Dataset
Amazon Sales Dataset (Kaggle) - contains product details, prices, ratings, and reviews.

## Visualizations Included
1. Bar Chart - Top 10 categories by number of products
2. Histogram - Distribution of product ratings
3. Pie Chart - Top 5 categories share of products
4. Histogram - Distribution of discounted prices
5. Scatter Plot - Discount percentage vs rating

## How to Run
1. Install required libraries: `pip install pandas matplotlib seaborn jupyter`
2. Open `vortextech_week1.ipynb` in Jupyter Notebook
3. Run all cells to see the analysis and visualizations
